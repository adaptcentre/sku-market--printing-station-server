### Config .env file

Make sure there is a config.env file in the root directory!

Config.env fields:

| Field name | Field value  | Comment |
--- | --- | ---
| NODE_ENV | `development or` `production` ||
| PORT | INTEGER ||

So th config file should look somehting like this:

```
NODE_ENV=
PORT=
```

```
node scripts/test-01.js
```
